# Interactive-quiz-app

An interactive and fun Quiz Application built using HTML, CSS, and JavaScript.  
This app tests users with multiple-choice questions, tracks time, calculates scores, and provides instant feedback — all wrapped in a clean and responsive UI.

Features

~ Multiple-choice quiz questions  
~ Countdown timer (5 minutes)  
~ Instant correct & wrong answer highlighting  
~ Automatic score calculation  
~ Restart quiz functionality  
~ Responsive and user-friendly design  

Technologies Used

~ HTML – Structure of the application  
~ CSS – Styling, layout, and responsiveness  
~ JavaScript – Quiz logic, timer, DOM manipulation, and event handling  

How It Works

1. User clicks Start to begin the quiz  
2. Questions appear one by one with four options  
3. User selects an answer and gets instant feedback  
4. Timer runs throughout the quiz  
5. Final score is displayed at the end  
6. User can restart the quiz anytime
